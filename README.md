# rust-columns

Columns implemented in Rust.

https://github.com/aoyama-val/rust-columns/assets/13144822/670195f6-6c02-436f-8ad5-3c1289041023


## Requirements

- SDL2
- Rust

## Controls

```
Left    : Move left
Right   : Move right
Down    : Drop
Space   : Rotate
Enter   : Restart when game over
```
